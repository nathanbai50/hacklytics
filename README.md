## AI Goal Logic

Goals are generated by sending the following data to the Python backend after each set:

- Total lifetime sets completed
- Scores and rep counts from the last 5 sets
- Average pushup depth across recent reps
- Coaching takeaways from recent sets

The backend returns a goal message, a rep target, and a score target. Goals only level up once the user has **met or exceeded** their current targets, ensuring progression feels earned.
